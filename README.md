<h1 align="center">Salah Times</h1>

<p align="center">
  A clean, reliable Islamic prayer times app for Android — built with Jetpack Compose.
</p>

---

## Features

- Accurate prayer times based on your location using the Aladhan API
- Fullscreen alarm that wakes your screen at each prayer time
- Athan sound plays automatically when the alarm triggers
- Reminder alarms before each prayer — timed intelligently based on prayer gaps
- Daily Quran verse on the home screen
- Smooth animations throughout
- Works fully in background with no battery drain

## Calculation Method

Defaults to Muslim World League (Method 3). Can be changed per-region in Settings.

## Permissions Required

- Location — to calculate accurate prayer times for your area
- Notifications — to show fullscreen prayer alarms
- Fullscreen Intent — to wake screen at prayer time
- Battery Optimization Exempt — to ensure alarms never miss

Built with GitHub Actions. Signed APK available in [Releases](../../releases).
